CORE VALUES

  • 追求卓越

    致力于打造“不断探索的进取型组织”

    对现状永怀积极、健康的不满,卓越的领导者一定是富有 企业家精神的，能点燃自己，更能点燃他人卓越的人善于把有价值的想法（气态）转化为易理解的说法（液态）和可执行的写法（固态），
    能简单明了但令人振奋地描述方向、愿景、目标和策略依靠人，也依靠制度来接近卓越追求长期可持续成功的精神状态。

  • 人文企业

    致力于打造“不断探索的进取型组织”

    以人的需求为尺度去丈量和设计我们的产品及服务， 并规范制度、流程以及经营行为；保持变革及危机意识，能迅速调整战略规划、组织形式、工作方法、思维方式来适应外部环境的变化，持续领跑；珍视和尊重员工的付出、贡献，并回馈员工更多；新力主张与社会分享价值，并承担卓越企业的社会义务。

  • 客户思维

    致力于打造“不断探索的进取型组织”

    所有的产品、服务、流程及相关制度都应围绕“为客户谋利益”（购买成本、使用成本、享用效果、保值增值等近期、长期利益）；客户的意见是我们的财富，以客户意见为标尺持续提升管理；衡量我们成功与否最重要的标准，是我们让客户满意的程度；对客户承诺的一定做到，没有承诺的，尽量做到；不是所有人都能成为我们的客户，但我们要用对待客户的思想对待所有人；客户思维同样适用于企业内部。下一道工序，就是你的客户。
